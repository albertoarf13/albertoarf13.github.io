Restaurant App Laravel

Link al proyecto: https://web-production-c373.up.railway.app/

Usuario: admin@admin.com

Contraseña: 12345678

Página web de un restaurante en la que puedes acceder como administrador y gestionar el menú, reservas de clientes, usuarios, etc. y también ver estadísticas de la empresa

La aplicación está hecha con PHP Laravel para el backend y MySql para la base de datos. Para crear el dashboard utilizamos un template llamado Concept, creado por colorlib.com


Podemos acceder a la aplicación como dueños o trabajadores del restaurante a través de este link: https://web-production-c373.up.railway.app/admin
